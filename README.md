# Finance Dashboard UI

This is a simple and interactive finance dashboard built using React and Tailwind CSS.

## Features

- Dashboard overview with total balance, income, and expenses
- Line chart for balance trend
- Pie chart for spending breakdown
- Transactions table with search, filter, and sorting
- Role-based UI (Admin / Viewer)
- Admin can add and delete transactions
- Insights section with key financial observations
- Dark mode support
- Data persistence using localStorage
- Export transactions as CSV
- Responsive design for all screen sizes

## Tech Stack

- React.js
- Tailwind CSS
- Recharts

npm install
npm start
